Proiect ierbar digital

Introducere

  1. Scopul produsului

    Scopul produsului este de a oferi utilizatorilor o aplicație web de tip ierbar digital, care să permită organizarea și gestionarea colecțiilor de plante presate și atributelor aferente. Aceasta va oferi o serie de facilități de căutare multi-criterială și va permite utilizatorilor să genereze, recomande și partajeze albume tematice, precum plante medicinale sau flori din zona montană.

  2. Audienta tinta

    Audienta vizata de aceasta aplicatie web de tip ierbar digital poate fi formata din persoane interesate de botanica, plante si flori, sau de utilizatori care doresc sa organizeze si sa-si gestioneze colectiile de imagini. De asemenea, aplicatia poate fi utila si pentru fotografi, mai ales pentru cei pasionati de natura si de fotografia de plante.

  3. Referințe

    • Unsplash API
    • W3Schools HTML
    • MDN Web Docs: Learn HTML

Descriere generală

  1. Descrierea produsului

    Această aplicație web oferă posibilitatea utilizatorilor de a organiza și gestiona colecțiile de plante. Utilizatorii pot căuta plante în funcție de diverse criterii, genera albume tematice și partaja colecțiile cu alți utilizatori. Aplicația oferă o interfață responsive si intuitiv de utilizat.

  2. Funcționalități produsului

    • Căutare multi-criterială a plantelor
    • Generarea de albume tematice
    • Generarea de rapoarte statistice despre plante
    • Clasamentul celor mai populare plante colectate

  3. Categorii de utilizatori

    • Colecționari de plante
    • Studenți sau cercetători în domeniul botanicii
    • Pasionați de plante și grădinărit

  4. Mediu de operare

    Aplicația este disponibilă pe platformele desktop, putând fi accesată prin intermediul unui browser web.

  5. Design și constrângeri de implementare

    Aplicația va fi dezvoltată folosind HTML, CSS și JavaScript, iar datele vor fi stocate într-o bază de date MongoDB. Interfața va fi creată cu ajutorul vanilla JS și va fi responsivă pentru a se adapta la diferite dimensiuni de ecran.

    Culori utilizate: white green black

  6. Dependențe

    • API-ul public Unsplash pentru a obține fotografii ale plantelor
    • Baza de date MongoDB pentru a stoca datele
    • JavaScript
    • NodeJS
    • HTML
    • CSS

Cerințele Interfeței

Pentru a asigura o utilizare facilă a produsului, interfața trebuie să îndeplinească următoarele cerințe:

  • Design simplu și intuitiv
  • Interfață responsivă și adaptabilă la diferite dispozitive
  • Posibilitatea de a filtra și căuta plante
  • Posibilitatea de a crea albume tematice
  • Posibilitatea de a vizualiza statistici despre colecție

  1. Interfețe Utilizator

    Interfața utilizator este formată din următoarele pagini:

    • Pagina principală - unde se pot căuta, filtra și vizualiza plante, și se pot accesa statistici despre colecție.
    • Pagina profil - unde se pot crea noi albume tematice, vizualiza imaginile apreciate, informatii despre utilizator, vizualizare cataloage
    • Pagina catolog - se pot vizualiza plantele de un anumit tip sau criteriu prin intermediul acestei pagini putem adauga fotografii in colectiile noastre
    • Pagina de vizualizare a statisticilor - unde se pot vizualiza diverse statistici despre colecție, cum ar fi cele mai populare plante.

  2. Interfețe Software

    Produsul va rula pe un server web creat cu NodeJS și va fi accesat prin intermediul unui browser web. Este compatibil cu următoarele browsere:

    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge
    • Safari
    • Etc
